Decorator et le composant Scheduler de Symfony

Le composant Scheduler de Symfony utilise le pattern Decorator pour modifier l'occurence d'une tâche planifiée.

Voici un exemple d'utilisation :

$trigger = new ExcludeTimeTrigger(new JitterTrigger(CronExpressionTrigger::fromSpec('#midnight', new MyMessage()));

$trigger->inner(); // CronExpressionTrigger
$trigger->decorators(); // [ExcludeTimeTrigger, JitterTrigger]

Dans cet exemple, une tâche planifiée est déclenchée à minuit, et est ensuite décorée pour être déclenchée de manière décalée (JitterTrigger) et exclure certaines dates (ExcludeTimeTrigger).
